# Claude Desktop + Telegraph MCP

## Setup

1. Copy the config below to your Claude Desktop MCP settings:
   - **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
   - **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

2. Replace `0xyour_key_here` with your EVM private key (funded with USDC).

3. Restart Claude Desktop. The Telegraph tools will appear in the tool list.

## Config

See [`claude_desktop_config.json`](./claude_desktop_config.json).
